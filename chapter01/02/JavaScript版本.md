# JavaScript版本

作为Netscape“继承人”的Mozilla公司，Mozilla 基金会继续开发JavaScript，添加新的特性、关键字和语法，JavaScript 的版本号继续递增。

目前，JavaScript 2.0 还没有目标实现。

大多数浏览器在提及对JavaScript 的支持情况时，一般都以ECMAScript 兼容性和对DOM的支持情况为准。
