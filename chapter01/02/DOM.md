# 文档对象模型（DOM）

文档对象模型（DOM），全称Document Object Model。是针对XML单晶硅扩展用于HTML的应用程序编程接口（API，Application Programming Interface）。

借助DOM提供的API，开发人员可以轻松自如地删除、添加、替换或修改任何节点。

**DOM并不只是针对JavaScript的，其他语言也都实现了DOM**
